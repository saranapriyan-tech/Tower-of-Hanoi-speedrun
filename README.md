# Tower of Hanoi Speedrun

The classic **Tower of Hanoi** puzzle with speedrunning features, online leaderboards, and a responsive user interface.

##  Live Demo

**Play here:** https://tower-of-hanoi-speedrun.vercel.app

---

##  About

Tower of Hanoi Speedrun is a browser-based puzzle game where players solve the Tower of Hanoi challenge as quickly as possible while minimizing moves.

The game supports multiple difficulty levels, tracks completion time and moves, and allows players to compete on online leaderboards.

---

##  Features

-  Interactive Tower of Hanoi gameplay
-  Fully responsive design
-  Choose between **3–12 disks**
-  Timer and move counter
-  Keyboard controls
-  Online leaderboards with **five categories**
-  Fast and lightweight interface

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/saranapriyan-tech/Tower-of-Hanoi-speedrun.git
cd Tower-of-Hanoi-speedrun
```

### Install dependencies

```bash
npm install
```

### Start the backend

```bash
npm start
```

Open index.html in your browser and you're ready to play.

---

##  How to Play
1. Enter your username
2. Select the number of disks (3–12).
3. Move one disk at a time.
4. Never place a larger disk on top of a smaller one.
5. Complete the puzzle in the fewest moves and fastest time.
6. Check your position in the leaderboards
